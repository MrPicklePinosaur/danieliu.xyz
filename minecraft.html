Installation Instructions

obv have minecraft installed

Next, go to this site and install minecraft forge 1.16.1 version 32.0.108. Make sure you download the installer.

Find the .jar filed you just downloaded and run it. Click install client and wait for it to do its ting.

NOTE: if you are not able to run the .jar, you probably need to install java

Now if you open up minecraft, you should see forge as an option to run, click play

On the main menu click on mods, and then click open mods folder on the bottom left corner

Download my modpack here and extract the file with your tool of choice. Now drag all the jar files into the folder that was opened

As an additional step, we also need to install the quark mod configuration file. Download it Here. Go back to the mod folder, and go BACK a directory to .minecraft

Find the folder called config and go to it. Now delete the file called quark-common.toml and replace it with the one you just downloaded.

After restarting minecraft you should be ready to play!
