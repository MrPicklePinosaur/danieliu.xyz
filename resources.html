Software I use

The common theme with software I use is that they are light, extensible, and (mostly) terminal based. Also, I like vim keys whenever and wherever possible :))

  • Arch Linux - I haven't really used other Linux distros so I can't comment too much on this

  • dwm - suckless window manager

  • dwmblocks - status bar

  • st - suckless terminal emulator

  • dmenu - application launcher and general purpose menu goodness

  • vim - my religion (you can find my vimrc here)

  • brave - web browser (chrome based, also comes with builtin ad-blocker)

  • zathura - pdf reader (doesn't read from Xresources for some reason so it's a bit annoying to customize)

  • sxiv - image viewer

  • mpv - video player (can also play videos from urls)

  • mpd + ncmpcpp - music

  • newsboat - rss reader

  • lf - file manager (this is basically ranger but not written in python)

  • btpd - bittorrent daemon

  • taskell - todo list

  • sxhkd - hotkeys

  • tiramisu + herbe - notifications
